### About this project
This is a simple data analysis project based on my personal data from Letterboxd, a social platform that I use to track, rate, and review the movies I watch. I requested my data from Letterboxd and used it to explore my movie-watching habits. The analysis was carried out using Pandas for data cleaning and manipulation and Matplotlib for data visualization. The goal of the project is to gain insights into my viewing patterns, such as the number of films watched, favorite genres, release years, ratings, and other trends reflected in my watch history.

## Project Objectives

* Clean and organize my exported Letterboxd data using Pandas.
* Analyze my movie-watching habits and preferences.
* Create visualizations to highlight interesting trends.
* Practice data analysis and visualization skills using Python.
